Télécharger l’IPA WeenoBis

Mini-guide iOS · WeenoBis

  1. Sur un PC Windows ou un Mac, télécharge et installe Sideloadly : Windows (64-bit), macOS · Windows 32-bit · site officiel
  2. Télécharge l’IPA ci-dessus — fichier WeenoBis.ipa (pas WeenoOff).
  3. Branche l’iPhone en USB, ouvre Sideloadly, glisse l’IPA, connecte-toi avec ton Apple ID (compte gratuit OK).
  4. Lance l’install. Sur l’iPhone : Réglages → Général → VPN et gestion de l’appareil → fais confiance au développeur.
  5. Ouvre WeenoBis. Compte maison : Wi‑Fi / VPN. Invité : colle le lien d’invitation wine-bis.

Important : sans Sideloadly (ou équivalent), l’IPA ne s’installe pas toute seule. À refaire ~tous les 7 jours avec un Apple ID gratuit (signature temporaire).

Après install : Wi‑Fi maison / VPN, ou lien d’invitation wine-bis en 4G/5G.

🤖 Android

Version : …

Fichier WeenoBis.apk — autorise les « sources inconnues » si Android le demande. Package distinct de Weeno (coexistent).

Télécharger l’APK WeenoBis

Mini-guide Android · WeenoBis

  1. Télécharge l’APK ci-dessus — fichier WeenoBis.apk (pas WeenoOff).
  2. Ouvre le fichier téléchargé. Si demandé, autorise Installer des apps inconnues pour ton navigateur / Fichiers.
  3. Valide l’installation (ou la mise à jour si tu as déjà WeenoBis). Pas besoin de désinstaller l’ancienne version WeenoBis.
  4. Ouvre WeenoBis. Compte maison : Wi‑Fi / VPN. Invité : lien d’invitation wine-bis.

Les mises à jour se font en téléchargeant la nouvelle APK par-dessus l’ancienne (même signature). Pas de Play Store. Weeno et WeenoBis peuvent coexister.

Cycle de maj indépendant de l’iPhone — compare avec la version dans l’app.
