🍎 iPhone · iPad

Version : …

Fichier WeenoOff.ipa — pas d’App Store. Installation via Sideloadly (PC ou Mac).

Télécharger l’IPA

Mini-guide iOS

  1. Sur un PC Windows ou un Mac, télécharge et installe Sideloadly : Windows (64-bit), macOS · Windows 32-bit · site officiel
  2. Télécharge l’IPA ci-dessus — fichier WeenoOff.ipa.
  3. Branche l’iPhone en USB, ouvre Sideloadly, glisse l’IPA, connecte-toi avec ton Apple ID (compte gratuit OK).
  4. Lance l’install. Sur l’iPhone : Réglages → Général → VPN et gestion de l’appareil → fais confiance au développeur.
  5. Ouvre Weeno. Compte maison : Wi‑Fi / VPN. Invité : colle le lien d’invitation dans l’app.

Important : sans Sideloadly (ou équivalent), l’IPA ne s’installe pas toute seule. À refaire ~tous les 7 jours avec un Apple ID gratuit (signature temporaire).

Après install : Wi‑Fi maison / VPN, ou lien d’invitation en 4G/5G.

🤖 Android

Version : …

Fichier WeenoOff.apk — autorise les « sources inconnues » si Android le demande.

Télécharger l’APK

Mini-guide Android

  1. Télécharge l’APK ci-dessus — fichier WeenoOff.apk.
  2. Ouvre le fichier téléchargé. Si demandé, autorise Installer des apps inconnues pour ton navigateur / Fichiers.
  3. Valide l’installation (ou la mise à jour si tu as déjà l’app). Pas besoin de désinstaller l’ancienne version — les données et le compte sont conservés.
  4. Ouvre l’app. Compte maison : Wi‑Fi / VPN. Invité : lien d’invitation.

Les mises à jour se font en téléchargeant la nouvelle APK par-dessus l’ancienne (même signature). Pas de Play Store.

Cycle de maj indépendant de l’iPhone — compare avec la version dans l’app.
